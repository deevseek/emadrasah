# E-Madrasah Agent Rules

- Kerjakan aplikasi sebagai modular monolith Laravel, bukan microservices.
- Gunakan controller tipis, Form Request, Policy/middleware permission, Service/Action, PHP Enum, Event/Listener/Job bila proses berat, transaction untuk proses kritis, dan automated testing.
- Jangan menyimpan kredensial rahasia, password produksi, token, atau file upload sebagai base64 di database.
- Jangan membuat angka dashboard palsu, grafik dummy, tombol tanpa fungsi, query di Blade, atau JavaScript sebagai satu-satunya validasi.
- Gunakan Bahasa Indonesia formal dan desain hijau tua dengan putih/abu muda serta emas hanya sebagai aksen terbatas.
