# Roadmap Implementasi

1. Fase 0: audit repository, dependensi, konfigurasi MySQL, arsitektur, ERD, deployment draft.
2. Fase 1: autentikasi, role, permission, profil madrasah, tahun ajaran, semester, pengaturan, audit, dashboard.
3. Fase 2: master akademik, siswa, wali, pegawai, kelas, mapel, jadwal, import/export.
4. Fase 3: absensi guru/siswa, izin guru, jurnal mengajar, approval.
5. Fase 4: BTAQ, penilaian, rapor, PDF.
6. Fase 5: tagihan, pembayaran, kuitansi, keuangan, payroll.
7. Fase 6: inventaris, program kerja, surat, pengumuman, kalender, laporan.
8. Fase 7: portal wali, notifikasi, WhatsApp provider abstraction, API, backup, health check.
9. Fase 8: hardening, audit N+1, audit permission, responsivitas, dokumentasi produksi.
