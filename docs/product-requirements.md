# Product Requirements E-Madrasah

Fase 0 menetapkan audit, konfigurasi MySQL, dokumentasi, dan pola arsitektur. Fase 1 mencakup autentikasi tanpa registrasi publik, akses berbasis role dan permission, profil madrasah, tahun ajaran/semester aktif tunggal, pengaturan aplikasi, layout backoffice, manajemen pengguna, audit log, seeder, dan automated test. Modul siswa, absensi operasional, BTAQ, pembayaran, keuangan, inventaris, dan fase berikutnya ditunda.
